
## หลักการจัดเก็บ

- แก้ไขเอกสารและภาพที่ใช้ร่วมกันผ่าน Pull Request
- เก็บ source code ของแต่ละทีมใน repo ของทีม หรือในโฟลเดอร์ที่ผู้สอนอนุมัติ
- ไม่อัปโหลดข้อมูลส่วนบุคคล ข้อมูลคำตอบแบบสอบถามจริง หรือ secret ใด ๆ
