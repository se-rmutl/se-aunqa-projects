# MVP Delivery Checklist

## Documentation

- [ ] Problem statement และ stakeholder
- [ ] SRS หรือ requirement summary
- [ ] Use Case Diagram
- [ ] Activity Diagram ของ workflow หลัก
- [ ] ER Diagram และ Data Dictionary
- [ ] Wireframe / UI flow

## Working Software

- [ ] MVP ทำงานได้อย่างน้อย 1 end-to-end workflow
- [ ] มี validation และ error handling ขั้นพื้นฐาน
- [ ] มี sample data ที่ไม่ใช่ข้อมูลจริง
- [ ] มี README สำหรับติดตั้ง/ใช้งาน

## Quality and Presentation

- [ ] Test cases และผลทดสอบ
- [ ] Demo video / live demo
- [ ] Slide หรือ poster
- [ ] ตัวอย่าง CSV/JSON Evidence Package
