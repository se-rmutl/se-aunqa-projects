## กติกาการทำงาน

- นัดประชุมอย่างน้อยสัปดาห์ละ 1 ครั้ง
- งานทุกชิ้นต้องมี Issue หรือ Task อ้างอิง
- Code ต้องทำผ่าน branch และ Pull Request
- การตัดสินใจสำคัญบันทึกใน Meeting Note

## Definition of Done

- ฟังก์ชันทำงานตาม Acceptance Criteria
- มี test case หรือหลักฐานทดสอบ
- เอกสารที่เกี่ยวข้องปรับปรุงแล้ว
- ผู้รับผิดชอบงาน review แล้ว
